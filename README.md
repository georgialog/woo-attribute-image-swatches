# WooCommerce Attribute Image Swatches

A simple WooCommerce plugin that replaces default attribute dropdowns with image-based attribute selection on the frontend.

## What it does

WooCommerce normally shows product attributes such as color, material, or style as standard select fields. This plugin lets those attributes be shown with images instead, so customers can choose variations visually.

It is useful for products where the appearance of an option matters more than its text label, such as colors, patterns, finishes, or styles.

## Features

- Displays product attribute options as images on the frontend.
- Helps customers understand variation choices faster.
- Works as a lightweight enhancement for WooCommerce variable products.
- Keeps the purpose focused: image-based attribute selection without unnecessary extras.

## Use case

Instead of asking customers to choose from a plain dropdown like:

- Red
- Blue
- Black

The plugin allows those options to be presented as visual image choices, which makes the selection process clearer and more user-friendly.

## Notes

This plugin was built to do one job: add image-based attribute selection for WooCommerce product attributes on the frontend.

There are no extra claims, bundled marketing features, or unnecessary complexity. It is a simple plugin that just works.
